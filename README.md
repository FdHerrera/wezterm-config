# WezTerm Config

Personal **WezTerm** configuration, designed to be portable across multiple unix-based machines.

## Installation

Clone the repository directly into WezTerm’s config directory:

```bash
git clone https://github.com/FdHerrera/wezterm-config $HOME/.config/wezterm
```

## Requirements
- Just (WezTerm)[https://wezterm.org/]
